# Netflix-DCF-Valuation
Equity valuation of Netflix using DCF, WACC analysis, and sensitivity modeling.
**Objective:** Estimate the intrinsic value of Netflix using a Discounted Cash Flow model.

**Key Components:**
- Revenue growth modeling & margin forecasting
- WACC and terminal value estimation
- Investment recommendation vs. current market price

**Result:**
Intrinsic Value per Share: $[210]
Market Price (as of date): $[540]
Recommendation: [Sell]

- DCF Excel Model
- Valuation Summary (PDF)
